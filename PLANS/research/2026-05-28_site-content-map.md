## 12. הצעת פלטה ל-Constitution §5

**Recommendation:** המיזוג של הפלטה הקיימת באתר החי + הכוונה ליוקרה (Constitution §1: "Luxury or nothing") יוצר את הפלטה הבאה — נשמור על האפרסק/קרם של האתר החי בכרגע, נחליק לכיוון אדום-עתיק יוקרתי במקום ה-`#CC3366` הזוועתי, ונעבה את הניגוד עם ink-deep:

```
--ink-deep:    #1A1410   /* near-black warm — replaces pure #000 */
--ivory:       #F5EFE6   /* warm cream — keep */
--brass:       #CFAE83   /* PRIMARY מ-live site, נשמר 1:1 */
--brass-deep:  #8B6F46   /* darker brass for hover/active */
--cocoa:       #534133   /* SECONDARY מ-live site, נשמר 1:1 */
--accent-rose: #B8576F   /* תחליף יוקרתי ל-#CC3366 הזוועתי של live */
--mist:        #E8DFD3   /* surface tint */
```

הסבר:
- `--brass #CFAE83` ו-`--cocoa #534133` הם **המקור האמיתי** של האתר החי. נשמרים בלי שינוי.
- `#CC3366` (קישורים/CTA פעילים) מוחלף ב-`#B8576F` — אותה משפחה אבל desaturated, נראה יוקרתי, פחות 90s-pink.
- `#000000` מוחלף ב-`#1A1410` (warm ink, כמעט שחור) — הניגוד נשמר 18:1+, אבל הוורמיות תואמת לפלטה הזו.
- `#FFFFFF` מוחלף ב-`#F5EFE6` (קרם) — מבטל את ה-"medical white" הזולה של live site, נותן ivory יוקרתי.

ראה ההחלטה הסופית ב-`CLAUDE.md` §5 (LOCKED 2026-05-28 by Agent 01).
